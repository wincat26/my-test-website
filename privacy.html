隱私權政策

感謝您使用本服務（GPT Actions for Google Calendar）。我們非常重視您的隱私與個人資料保護，請詳細閱讀本政策，以了解我們如何蒐集、使用與保護您的資訊。

1. 資料蒐集與用途

  • 本服務僅於串接 Google Calendar API 過程中，取得您的行事曆授權資訊。
  • 所有授權均經由 Google OAuth2 驗證，服務本身不會永久儲存您的任何敏感資料。
  • 蒐集之資料僅用於提供日曆查詢、事件建立與管理等功能，不會用於其他商業用途。

2. 第三方服務

  • 本服務透過 OpenAI ChatGPT Actions 及 Google API 提供功能，使用時請同時參閱 Google 隱私權政策 及 OpenAI 隱私權政策。

3. 資料保護

  • 所有通訊過程均採用加密傳輸（HTTPS）。
  • 您的授權資料僅於執行操作時暫時使用，不會主動存儲或提供予第三方。

4. 使用者權利

  • 您可以隨時撤銷本服務對您 Google 帳戶的存取權限（請至 Google 帳戶設定 管理）。
  • 如對本政策有疑問，可隨時與我們聯絡。

5. 聯絡我們
